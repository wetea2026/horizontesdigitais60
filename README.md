# Horizontes Digitais 60+
> **Workshop de Educação, Tecnologia e Envelhecimento Ativo**  
> *Evento Integrado ao CBIE 2026 (Congresso Brasileiro de Informática na Educação)*

Este é o repositório oficial do website do **Horizontes Digitais 60+**. O projeto foi desenvolvido com foco total em acessibilidade, usabilidade e design responsivo, servindo como a central de informações para chamadas de trabalhos, cronogramas e organização do workshop.

## 🔗 Demonstração Disponível (Live Demo)
O site está publicado e pode ser acessado diretamente através do GitHub Pages:
https://wetea2026.github.io/horizontesdigitais60/

---

## 📁 Estrutura de Diretórios Otimizada

O projeto segue uma arquitetura modular e limpa para facilitar manutenções futuras:

```text
├── assets/
│   ├── css/
│   │   └── style.css            # Folha de estilos global, variáveis e alto contraste
│   ├── js/
│   │   └── script.js            # Lógica de acessibilidade, barra de progresso e scroll
│   └── images/
│       ├── logo.png             # Ícone quadrado da navbar
│       ├── org/                 # Fotos dos coordenadores gerais
│       ├── sections/            # Imagens laterais das seções (Sobre, Datas, etc.)
│       ├── support/             # Logos institucionais (Realização e Apoio)
│       └── favicon/             # Ícones de navegador
├── index.html                   # Página principal do Workshop
